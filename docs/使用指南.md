# 台灣癌症登記資料自動化解碼系統
## 乳癌轉移性癌症資料庫分析指南

---

## 📋 系統概述

本系統的目標是將你的 **129 個編碼欄位** 轉換為 **可理解的臨床敘述**，並構建每個患者的 **臨床故事**。

### 核心問題解決
- ✅ **解碼問題**：編碼值 → 臨床描述
- ✅ **信息提取**：從 129 欄位中篩選出關鍵臨床指標
- ✅ **臨床故事**：按診斷→治療→預後串連資訊
- ✅ **亞群分層**：按組織學、TNM、激素受體狀態等自動分類

## 🛠️ 實施方案

### 方案 A：使用 Excel 內建公式（最簡單）

#### 步驟 1：建立欄位字典

在你的 Excel 中新增一個 Sheet：**「欄位字典」**

| 欄位名稱 | 編碼值 | 臨床描述 | 資料來源 |
|---------|-------|--------|--------|
| TNM_T | 0 | Tis (原位癌) | 114年版長表 P.xx |
| TNM_T | 1 | T1 (≤20mm) | 114年版長表 P.xx |
| TNM_T | 2 | T2 (>20mm, ≤50mm) | 114年版長表 P.xx |
| TNM_N | 0 | N0 (無淋巴結轉移) | 114年版長表 P.xx |
| ... | ... | ... | ... |

**資料來源**：
- `Longform-Manual_Official-version_20251224_W-1.pdf` (第129欄位定義)
- `Cancer-SSF-Manual_Official-version_20251204_W.pdf` (乳癌特定因子)

#### 步驟 2：建立解碼公式

在原始資料旁新增「解碼」欄位，使用 VLOOKUP：

```excel
=VLOOKUP([@TNM_T], 欄位字典!$A$2:$C$999, 3, 0)
```

將所有編碼欄位都加上相應的 VLOOKUP 公式。

#### 步驟 3：建立臨床資訊摘要

新增 Sheet：**「患者臨床摘要」**

每個患者一列，包含：
```
患者ID | 年齡 | 性別 | 診斷日期 | 
原發位置 | 組織學類型 | TNM分期 | 分化程度 | 
激素受體 | HER2狀態 | 手術類型 | 化療方案 | 
轉移部位 | 復發情況 | 隨訪時間 | 存活狀態
```

使用 VLOOKUP 自動填入解碼後的值。

#### 步驟 4：建立亞群分類

新增 Sheet：**「亞群分析」**

使用 COUNTIF 統計各亞群的患者數：

```excel
激素受體陽性 (ER+/PR+):  =COUNTIF(患者摘要!$E$2:$E$999, "ER+/PR+")
激素受體陰性 (ER-/PR-):  =COUNTIF(患者摘要!$E$2:$E$999, "ER-/PR-")
HER2陽性:              =COUNTIF(患者摘要!$F$2:$F$999, "HER2+")
TNM I期:               =COUNTIF(患者摘要!$D$2:$D$999, "*I期*")
有遠距轉移:             =COUNTIF(患者摘要!$L$2:$L$999, "*轉移*")
有局部復發:             =COUNTIF(患者摘要!$M$2:$M$999, "*復發*")
```

### 方案 B：使用 Python 自動化（更強大）

#### 環境需求
```bash
pip install pandas openpyxl
```

#### 簡化版腳本

```python
import pandas as pd

# 1. 讀取資料
df_raw = pd.read_excel("metaplastic carcinoma data base.xlsx")
df_dict = pd.read_excel("癌症登記資料解碼系統.xlsx", sheet_name="欄位字典")

# 2. 建立對照表
# 對每個需要解碼的欄位，使用字典進行映射
for field in df_raw.columns:
    if field in df_dict['欄位名稱'].unique():
        mapping = dict(zip(
            df_dict[df_dict['欄位名稱']==field]['編碼值'],
            df_dict[df_dict['欄位名稱']==field]['臨床描述']
        ))
        df_raw[f'{field}_解碼'] = df_raw[field].map(mapping)

# 3. 構建臨床故事
clinical_summary = df_raw[[
    '患者ID', '年齡', '性別',
    'TNM_T_解碼', 'TNM_N_解碼', 'TNM_M_解碼',
    '組織學_解碼', '分化程度_解碼',
    '激素受體_解碼', 'HER2_解碼',
    '手術類型_解碼', '化療方案_解碼',
    '轉移_解碼', '復發_解碼'
]].copy()

# 4. 亞群分類
subgroups = {
    'ER+/PR+': (clinical_summary['激素受體_解碼'] == 'ER+/PR+').sum(),
    'ER-/PR-': (clinical_summary['激素受體_解碼'] == 'ER-/PR-').sum(),
    'HER2+': (clinical_summary['HER2_解碼'].str.contains('+')).sum(),
    '三陰性': ((clinical_summary['激素受體_解碼'] == 'ER-/PR-') & 
              (clinical_summary['HER2_解碼'] == 'HER2-')).sum(),
}

# 5. 匯出結果
with pd.ExcelWriter("癌症登記資料_解碼結果.xlsx", engine='openpyxl') as writer:
    df_raw.to_excel(writer, sheet_name="原始+解碼", index=False)
    clinical_summary.to_excel(writer, sheet_name="臨床摘要", index=False)
    pd.DataFrame(subgroups.items(), columns=['亞群', '患者數']).to_excel(
        writer, sheet_name="亞群統計", index=False
    )
```

## 📊 欄位對照表 (乳癌常見編碼)

### TNM 分期
| 欄位 | 編碼 | 臨床含義 |
|-----|------|--------|
| T | 0 | Tis (原位癌) |
| T | 1 | T1 (≤20mm) |
| T | 2 | T2 (>20mm, ≤50mm) |
| T | 3 | T3 (>50mm) |
| T | 4 | T4 (胸壁/皮膚侵犯) |
| N | 0 | N0 (無淋巴結轉移) |
| N | 1 | N1 (腋下1-3個) |
| N | 2 | N2 (固定淋巴結) |
| N | 3 | N3 (內乳淋巴結) |
| M | 0 | M0 (無遠距轉移) |
| M | 1 | M1 (有遠距轉移) |

### 組織學類型
| 編碼 | 臨床含義 |
|-----|--------|
| 1 | 浸潤性導管癌 (IDC) |
| 2 | 浸潤性小葉癌 (ILC) |
| 3 | 特殊類型 (膜狀癌、髓樣癌等) |
| 4 | 轉移性癌 |

### 激素受體狀態
| 編碼 | 臨床含義 |
|-----|--------|
| 1 | ER+/PR+ (Luminal, 激素敏感) |
| 2 | ER+/PR- (Luminal, 激素敏感) |
| 3 | ER-/PR+ (Luminal, 罕見) |
| 4 | ER-/PR- (非Luminal) |

### HER2 狀態
| 編碼 | 臨床含義 |
|-----|--------|
| 0 | 陰性 (0/1+) |
| 1 | 陽性 (3+或 FISH+) |
| 2 | 等陽性 (2+，需FISH確認) |

### 分化程度 (Nottingham Grade)
| 編碼 | 臨床含義 |
|-----|--------|
| 1 | Grade I (高分化) |
| 2 | Grade II (中分化) |
| 3 | Grade III (低分化) |

## 🎯 關鍵臨床指標 (Subgroup)

### 分子亞型分類
```
1. Luminal A: ER+/PR+, HER2-, Ki67低
2. Luminal B: ER+或PR+, HER2+或Ki67高
3. HER2-enriched: ER-/PR-, HER2+
4. 三陰性 (TNBC): ER-/PR-/HER2-
```

### TNM 分期亞組
```
I期: T1, N0, M0
II期: T0-2, N1, M0 或 T3, N0, M0
III期: T0-3, N2-3, M0 或 T4, 任何N, M0
IV期: 任何T, 任何N, M1
```

### 臨床結局亞組
```
- 有無遠距轉移 (M0 vs M1)
- 有無局部復發
- 有無淋巴結轉移 (N0 vs N+)
```

## 💡 建議優先順序

1. **第1優先**：TNM分期 (診斷的基礎)
2. **第2優先**：組織學類型 + 分化程度
3. **第3優先**：激素受體 + HER2 (預後和治療決定)
4. **第4優先**：治療方案 (手術、化療、放療)
5. **第5優先**：轉移和復發狀況
6. **第6優先**：其他預後指標

## 🔗 後續分析建議

解碼完成後，可進行：
1. **臨床特徵相關性分析** - 分子亞型與TNM、生存期的關係
2. **治療模式分析** - 不同亞群的治療選擇
3. **預後分析** - 轉移和復發的危險因子
4. **生存曲線** - 按亞群繪製Kaplan-Meier曲線
